# 3x3 퍼즐 맞추기 프로그램 (C++)

이 프로젝트는 9개의 퍼즐 조각을 3x3 보드에 올바르게 배치하는 프로그램입니다.
각 조각은 4방향(0, 90, 180, 270도)으로 회전이 가능하며, 인접한 조각의 면끼리 값의 합이 0이 되도록 맞춥니다.

---

## 📌 주요 기능

* 9개 퍼즐 조각의 3x3 자동 배치
* 각 조각별 4방향(회전) 지원
* 인접면이 모두 조건(합=0)을 만족하는 모든 해 출력
* 정답이 없을 경우 안내 메시지 출력

---

## 🧠 사용한 알고리즘

### 🧩 백트래킹(Backtracking)

* 모든 칸에 대해 조각과 회전 방향을 하나씩 선택
* 배치 중 인접(위/왼쪽) 조각과의 합이 0이 아닐 경우 즉시 가지치기(탐색 중단)
* 모든 가능한 정답을 효율적으로 탐색

---

## 📁 출력 결과

* 모든 해(`정답 #N`)를 콘솔에 출력
  예시:

  ```
  === 정답 #1 ===
  1(0도) 2(90도) 3(180도)
  4(270도) 5(0도) 6(90도)
  7(180도) 8(270도) 9(0도)
  ```
* 정답이 없을 때:

  ```
  정답이 존재하지 않습니다.
  ```

---

## 🛠 실행 방법

### 1. 컴파일

```bash
g++ -o puzzle_solver main.cpp
```

### 2. 실행

```bash
./puzzle_solver
```
